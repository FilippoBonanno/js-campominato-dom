# CampoMinato

**CampoMinato** è un semplice gioco del campo minato, sviluppato con **HTML**, **CSS**, **JavaScript** e **Bootstrap**. L'utente deve cercare di ottenere il punteggio più alto possibile evitando di cliccare sulle caselle che nascondono bombe.

## Tecnologie Utilizzate

- **HTML5**: Struttura di base dell'applicazione.
- **CSS3**: Per lo stile del layout e dell'interfaccia.
- **JavaScript (ES6)**: Per la logica di gioco dinamica e le interazioni.
- **Bootstrap 5.3.3**: Per la creazione di un layout reattivo e moderno.

## Funzionalità Principali

- **Griglia Interattiva**: Griglia di gioco creata dinamicamente con celle che possono nascondere bombe.
- **Punteggio Dinamico**: Il punteggio viene aggiornato ogni volta che l'utente clicca su una cella sicura.
- **Conteggio Bombe**: Il numero di bombe presenti viene visualizzato e aggiornato.
- **Reset Partita**: Un pulsante permette di riavviare il gioco in qualsiasi momento.

## Lezioni Apprese

Durante lo sviluppo di questo progetto, ho imparato a:
- Utilizzare **JavaScript** per gestire dinamicamente elementi del DOM.
- Implementare la logica di un gioco interattivo.
- Gestire eventi e aggiornamenti dello stato dell'interfaccia in tempo reale.
- Creare layout responsivi utilizzando **Bootstrap**.
